# Home Apothecary Website - Page Layouts & Design Concepts

## 🎨 Design System

### Color Palette
**Primary Colors (Natural & Herbal Theme):**
- **Sage Green**: #7C9885 (Primary brand color)
- **Deep Forest**: #3D5A40 (Dark accent)
- **Cream**: #F5F1E8 (Background)
- **Terracotta**: #D4816B (Call-to-action)
- **Warm White**: #FDFDF9 (Cards/panels)

**Secondary Colors:**
- **Amber**: #E8A87C (Warnings/cautions)
- **Soft Blue**: #8FA8C7 (Info elements)
- **Charcoal**: #2C3333 (Text)
- **Light Gray**: #E8E8E8 (Borders)

### Typography
- **Headings**: Playfair Display or Cormorant (Serif, elegant)
- **Body**: Inter or Source Sans Pro (Sans-serif, readable)
- **Accent**: Raleway (Clean, modern)

### Spacing
- Base unit: 8px
- Section padding: 64px (desktop), 32px (mobile)
- Component spacing: 16px, 24px, 32px

## 📱 Mobile Responsive Considerations

### Mobile Layout Changes:
- **Navigation**: Hamburger menu
- **Video**: Full-width, portrait-optimized
- **Cards**: Single column stack
- **Comments**: Simplified layout
- **Forms**: Touch-optimized inputs
- **Tabs**: Swipeable horizontal scroll

### Touch Interactions:
- Swipe for recipe carousel
- Pull to refresh
- Tap to expand/collapse sections
- Long press for options menu

---

## ♿️ Accessibility Features

- Semantic HTML5 elements
- ARIA labels for screen readers
- Keyboard navigation support
- Focus indicators
- Alt text for all images
- Captions for videos
- High contrast mode option
- Font size adjustment
- Skip to content link

---

## 🎬 Animations & Interactions

### Subtle Animations:
- Fade in on scroll (for recipe cards)
- Smooth transitions (0.3s ease)
- Hover effects (scale 1.05, shadow increase)
- Loading skeletons
- Toast notifications
- Modal slide-ins
- Progress indicators

### Page Transitions:
- Fade between pages
- Slide for nested navigation
- Modal overlays with backdrop blur

### On Dashboard:
- Recipes saved
